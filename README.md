# OBIXCONFIG FPV

A premium, mobile-first FPV website concept built as a clean static repo.

## What is inside
- `index.html` — main landing page
- `styles.css` — visual system and responsive layout
- `app.js` — tool selection, menu toggle, reveal animation, PWA registration
- `manifest.webmanifest` — PWA manifest
- `sw.js` — basic offline cache
- `assets/` — logo and social preview assets

## How to run
Open `index.html` directly in a browser, or serve the folder with any static server.

Example:
```bash
python -m http.server 8080
```

Then visit:
```text
http://localhost:8080
```
